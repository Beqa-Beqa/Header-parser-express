## Set up on your local machine

```bash

git clone https://github.com/Beqa-Beqa/Header-parser-express.git #This will clone the repository on your local machine

npm install #installs neccessary dependencies on your machine

npm start #Runs nodemon that will look for index.js file changes. It will also let you test the app on your local host

```

Make a request on [host]/api/whoami to see the details about your browser headers
